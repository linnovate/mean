## Welcome to the mean stack

The mean stack is intended to provide a simple and fun starting point for cloud native fullstack javascript applications.

### Installation 
``` 
cp .env.example .env
yarn install
yarn build
yarn start (for development)
yarn serve (for production)
```
### Docker based 
``` 
cp .env-example .env
docker-compose up -d
```
